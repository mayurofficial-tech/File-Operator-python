# 📓 File Operator - Personal Journal Manager

## 📌 Description
File Operator is a Python-based console application that allows users to manage personal journal entries using file handling.

Users can add, view, search, and delete journal entries stored in a text file.

---

## 🚀 Features
- ✍️ Add journal entry with timestamp
- 📖 View all entries
- 🔍 Search entries by keyword
- ❌ Delete journal file
- ⚠️ Exception handling for file operations

---

## 🛠️ Technologies Used
- Python
  - File Handling
  - OOP (Classes & Methods)
  - Exception Handling
  - Date & Time module

---

## 📂 Concepts Covered
- File operations (read, write, append)
- Try-except blocks
- String manipulation
- Menu-driven program
